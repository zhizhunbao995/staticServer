
# staticServer

  Simple server.

## Installation

    $ npm install -g staticServer

## Usage

```
Usage: static [options] 

Options:
  -h, --help                output usage information
  -V, --version             show version number
  -p, --port <port>         specify the port [3000]
  -c  --autoindex off       if set off, request folder return 403
```

## Examples

```
static -p 8080
```
